# todo list
## 소개
스프링 부트와 리액트로 만든 todo list 앱
## 빌드환경
- jdk 1.8
- Maven 3.5.3
## 빌드방법
1. 터미널 접속
2. git clone https://github.com/SehwaKim/todo-list.git
3. git clone이 끝나면 clone된 프로젝트로 이동(cd todo-list) 
4. mvn install
5. java -jar target/todo-list-0.0.1-SNAPSHOT.jar
6. localhost:8080 으로 접속
